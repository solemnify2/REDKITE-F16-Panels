Protocol Reference — DCS-BIOS & BMS-BIOS LED Mapping
=====================================================


1. DCS-BIOS (DCS World F-16C_50)
=================================

Source: DCS-BIOS F-16C_50.json
  Path: %USERPROFILE%\Saved Games\DCS\Scripts\DCS-BIOS\doc\json\F-16C_50.json
  Web:  http://localhost:5010/reference/  (DCS 실행 중)

Protocol:
  - Binary, sync 0x55 x4
  - Repeating chunks: ADDR_LO ADDR_HI COUNT_LO COUNT_HI DATA[count]
  - Frame end: addr=0x5555, count=0x5555
  - Baud: 1,000,000

2. BMS-BIOS (Falcon BMS 4.38)
===============================

Source: Falcon BMS 4.38 FlightData.h
  Path: C:\Falcon BMS 4.38\Tools\SharedMem\FlightData.h

Protocol:
  - Binary, sync 0xAA 0xBB
  - Payload: ledBits (uint32 LE, 4 bytes)
    - Bits 0–10: LED states (see LED Index Mapping)
    - Bit 16: Backlight (1=ON from instrLight)
  - Checksum: XOR of 4 payload bytes (1 byte)
  - Total: 7 bytes per frame
  - Baud: 1,000,000

Shared Memory:
  - Name: "FalconSharedMemoryArea"
    FlightData struct offsets:
      lightBits  = 108  (unsigned int)
      lightBits2 = 124  (unsigned int)
      lightBits3 = 128  (unsigned int)
  - Name: "FalconSharedMemoryArea2"
    FlightData2 struct offsets:
      instrLight = 1104  (unsigned char, 0=OFF, 1=DIM, 2=BRT)

LightBits Mask Map (used in bmsbios_bridge.py):

  LightBits (offset 108):
    Name            Mask(hex)     Bit  Description
    -------------   ----------    ---  -----------
    AdvStandby      0x80000000     31  ADV Standby indicator

  LightBits2 (offset 124):
    Name            Mask(hex)     Bit  Description
    -------------   ----------    ---  -----------
    AuxSrch         0x00001000     12  TWA Search
    AuxAct          0x00002000     13  TWA Activity
    AuxLow          0x00004000     14  TWA Alt Low
    AuxPwr          0x00008000     15  TWA Power
    EcmPwr          0x00010000     16  ECM power indicator
    AdvActive       0x20000000     29  ADV Active indicator
    GEARHANDLE      0x40000000     30  Gear handle lamp (gear in motion)

  LightBits3 (offset 128):
    Name            Mask(hex)     Bit  Description
    -------------   ----------    ---  -----------
    NoseGearDown    0x00010000     16  Nose gear down and locked
    LeftGearDown    0x00020000     17  Left gear down and locked
    RightGearDown   0x00040000     18  Right gear down and locked

4. Protocol Auto-Detection
===========================

  Teensy receives serial data and auto-detects protocol:
    - 0x55 x4 consecutive  →  DCS-BIOS
    - 0xAA then 0xBB       →  BMS-BIOS
  Timeout (6 sec no data)  →  reset to UNKNOWN, re-detect
